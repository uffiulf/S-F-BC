## 4. Økonomisk beredskap

| Scenario | Handling |
|----------|----------|
| Får ikke aksjekapital | Stift som forening i stedet (lavere kostnad, høyere risiko). Alternativt: utsett AS-stiftelse og jobb videre med planlegging til midlene er på plass |
| Får ikke driftstilskudd | Reduser kostnader: gratis verktøy, frivillige mentorer, hjemmekontor |
| Må betale ESA-avklaring selv | Søk pro bono-bistand fra jusstudenter eller advokatfirmaer med samfunnsansvar |
| Kundene kan betale noe | Vurder symbolsk betaling (f.eks. 80 kr/time) for å dekke forsikring og drift — se tilpasningsforslag 3B |
