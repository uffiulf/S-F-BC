# Gjennomføringsplan — Syntax & Flow
## 4 faser fra «minst motstand» til «mest motstand»

> **Mål:** Realisere konseptet med *minst mulig juridisk og operasjonell motstand*.
> Bygger på juridisk vurdering (juridisk_vurdering_og_risikoanalyse.md), utdyping av statsstøtte (utdyping_statsstotte_eos.md), forretningsmodell, GTM-strategi, markedsanalyse, risikoanalyse og organisasjonsdokumenter.
>
> **Utarbeidet:** 18. juni 2026 | **Versjon:** 1.0

---

## Sammenheng mellom risiko og faser

| Risiko (fra juridisk vurdering) | Alvorlighet | Adresseres i fase |
|---|---|---|
| 1. Lånekassen godkjenner ikke støtte | **Kritisk** | Fase 1 + 2 |
| 2. Statsstøtte/EØS – HiØs ressurser | **Kritisk** | Fase 2 (må være løst *før* Fase 3) |
| 3. Feilklassifisering av studenter | **Høy** | Fase 2 |
| 4. Konkurranseklager | **Høy** | Fase 2 + 3 + 4 |
| 5. Ansvarsforhold / forsikring | **Høy** | Fase 2 |
| 6. Personvern (GDPR) / portal | **Middels** | Fase 2 + 3 |
| 7. Skatt/MVA/org.form | **Middels** | Fase 2 |

### Overordnet prinsipp

```
Fase 1: Bare prat og planer — null risiko
Fase 2: Rydde juridisk og organisatorisk — før noe leveres til kunde
Fase 3: Ett prosjekt — test og dokumenter
Fase 4: Skaler — med erfaring og sikkerhet fra fase 3
```

**Ingen kundeleveranser før Fase 3. Ingenting offentliggjøres før Fase 2 er fullført.**

## Hva gjør dere hvis...

| Scenario | Handling |
|---|---|
| **Lånekassen sier NEI** | Redesign emnet (f.eks. legg det inn som siste semester i graden, eller søk unntak). Alternativt: Spor 3 (NAV) som hovedspor, eller stiftelsesmidler. |
| **HiØ sier NEI til samarbeidsavtale** | Start uten HiØ — som ren studentforening. Men da mister dere mentorordningen og merkevaren. Vurder om modellen likevel kan fungere. |
| **ESA-klage kommer** | Ikke få panikk. Svar innen fristen. Bruk «defense pack» (4.5). Vis frem at alt er dokumentert, at HiØ-bidrag er priset til markedspris, og at hver kunde er under de minimis. |
| **Student slutter midt i prosjekt** | Standardavtalen med kunden sier at det ikke er noen leveransegaranti. Sett inn reserve fra teamet. Dokumentér. |
| **Kunde klager på kvalitet** | Minn på avtalen (gratis, best-effort). Tilby retting hvis mulig. Lær og justér. |
| **NAV-deltaker får helseproblemer** | Følg NAVs retningslinjer. Syntax & Flow er vertsvirksomhet (ikke arbeidsgiver). Kontakt NAV-veileder umiddelbart. |

## Kritiske forutsetninger

1. **Lånekassen må si JA** til støtte for bedriftspraksis etter fullført grad. Uten dette — stopp.
2. **HiØ må godkjenne emnet** formelt og inngå samarbeidsavtale med markedspris. Uten dette — ulovlig statsstøtte.
3. **Ingen kundeleveranser før Fase 3.** Alt annet er uforsvarlig.
4. **Hold dere unna kompliserte oppdrag.** Nettsider, logoer, enkle analyser — ikke ERP-systemer, ikke regnskap for andres regnskap, ikke personopplysningssystemer.
5. **Dokumentasjon, dokumentasjon, dokumentasjon.** Uten dokumentasjon har dere intet forsvar.
