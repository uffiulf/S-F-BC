# HiØ — Innovasjonsmidler og støtteordninger for studentinitiativer

> **Research utført:** 22. juni 2026  
> **Formål:** Kartlegge interne støtteordninger ved Høgskolen i Østfold for Syntax & Flow

## 1. Interne midler for studentprosjekter

**❌ IKKE BEKREFTET:** Ingen dedikerte studentinnovasjonsmidler funnet på hiof.no.  
HiØs interne finansieringsordninger er primært rettet mot **ansatte/forskere**, ikke studenter:

| Ordning | Målgruppe | Status |
|:--------|:----------|:-------|
| Strategiske midler — "Det digitale samfunn" | Forskere/ansatte | ✅ Bekreftet |
| Forskningsløftet | Forskere/ansatte | ✅ Bekreftet |
| Fakultetsmidler (HVO, LUSP, IIØ) | Ansattes reiser/konferanser | ✅ Bekreftet |
| Publiseringsfond | Ansattes publiseringsavgifter | ✅ Bekreftet |
| **Studentinnovasjonsmidler** | **Studenter** | **❌ Ikke funnet** |

**Kilde:** [hiof.no/forskning/forskningsstotte/intern-finansiering/](https://www.hiof.no/forskning/forskningsstotte/intern-finansiering/)

## 5. Samarbeid med SiØ (Studentsamskipnaden i Østfold)

**❌ IKKE BEKREFTET:** Ingen spesifikt innovasjonssamarbeid funnet mellom SiØ og studentbedrifter.

SiØs rolle er primært:
- Studentbolig, spisesteder, helse- og rådgivningstjenester
- Støtte til studentforeninger (FSS/HSS) gjennom aktivitetsmidler
- **Ingen dedikerte innovasjons- eller start-midler**

**Kilde:** [SiØ lag og foreninger](https://www.siost.hiof.no/studentmilj%C3%B8/lag-og-foreninger)

**Anbefalte neste steg:**
1. Ta direkte kontakt med **Bjørn Klimek** (bjornkl@hiof.no) for å høre om uformelle ordninger
2. Kontakt **Studentparlamentet** for å kartlegge studentdrevne initiativer
3. Sjekk HiØs **intranett** (krever innlogging) — kan ha interne sider om studentinnovasjon
4. Vurder eksterne kilder: **SIVA, Innovasjon Norge, Smart Innovation Norway** (se egne forskningsdokumenter)
