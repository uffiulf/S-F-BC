# Syntax & Flow - Nettside og Infoportaler

Denne mappen inneholder kildekoden til våre web-applikasjoner:

1. **`syntax-flow-app/`**: Vår hovedside og nettportal bygget med Next.js, React og Tailwind CSS.
2. **`infoside-app/`**: En interaktiv infoportal og retningslinje-side for teamet, bygget med ren HTML, CSS og JavaScript (har mørk/lys modus og en avansert søkemotor).

## Hvordan kjøre lokalt:

### 1. Infoside
Åpne `07-Nettside/infoside-app/index.html` direkte i nettleseren din eller bruk en enkel HTTP-server (f.eks. Live Server i VS Code).

### 2. Hovedside (syntax-flow-app)
Gå til `07-Nettside/syntax-flow-app/` og kjør:
```bash
npm install
npm run dev
```
