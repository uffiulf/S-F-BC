# Syntax & Flow
> Et brobyggende rammeverk for arbeidserfaring etter fullført utdanning ved Høgskolen i Østfold (HiØ).

**Syntax & Flow** er en studentdrevet bedrift etablert som et ledd inn i **Høgskolen i Østfold**. Formålet er å gi studenter innen **Økonomi, IT og Design** (både på bachelor- og masternivå) verdifull og lønnet arbeidserfaring etter endt studieløp. Dette gjøres gjennom et skreddersydd fag/program i **Bedriftspraksis**, der studentene utfører reelle oppdrag for næringslivet mens de mottar støtte/lønn via Lånekassen.

I dette prosjektet er **studenten selv det endelige produktet** for både høgskolen (som tiltrekker aktive søkere) og næringslivet (som får en risikofri test-run før ansettelse).
